Projects

Alzheimer's Disease Prediction Using Deep Learning Algorithms

Overview

As part of a four-member academic capstone project, we developed a deep learning model to predict and stage Alzheimer's Disease using MRI brain images. The goal was to create a reliable, automated system for accurately identifying the various stages of Alzheimer's to aid in early diagnosis and treatment.

Roles and Contributions

  • Model Design: Contributed to the design and implementation of the Convolutional Neural Network (CNN) for image classification. Assisted in selecting the architecture and optimizing its performance.

  • Data Management: Played a key role in processing and analyzing a dataset of 6,400 MRI images categorized into four stages: Non-Demented, Very Mild Demented, Mild Demented, and Moderate Demented. Applied data augmentation techniques to improve model robustness.

  • Technologies Utilized: Utilized Python, TensorFlow, and Keras for model development, and OpenCV for preprocessing MRI images. Employed Jupyter Notebook for development, experimentation, and result visualization.

  • Validation and Testing: Assisted in validating and testing the model to ensure accuracy and reliability, analyzing performance metrics to help refine the model.

  • Team Collaboration: Actively collaborated with team members, contributing to various aspects of the project, sharing insights, and participating in problem-solving efforts.

Key Achievements

  • Accuracy: Contributed to achieving an accuracy of 97.04% in classifying Alzheimer's Disease stages, demonstrating the model’s high effectiveness.

  • Data Augmentation: Implemented advanced augmentation techniques such as rotation, zoom, and flip to enhance the model’s generalization capabilities.

  • Practical Impact: Showcased the potential of deep learning in solving complex healthcare problems and demonstrated readiness to apply these skills in practical scenarios.

Impact

This project illustrates the application of deep learning to address significant healthcare challenges and reflects my technical abilities, collaborative skills, and capacity to contribute to impactful technological solutions.
